## Install prerequisites

### node
```
brew install node
```

### npm (create package.json)
```
$ mkdir test
$ cd test
$ npm init
```

### npm install library
```
$ npm install [lib-name]
$ npm install [lib-name] --save-dev

$ npm uninstall [lib-name]

$ npm run [script-name]

// typescript
$ npm install typescript @types/node
$ npm install ts-node

// changes automatically reflection
$ npm install nodemon
```
